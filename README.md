Basic Remote Desktop Implementation (C#)

Project Overview: This project is an educational prototype designed to understand the basics of network programming. It establishes a Client-Server connection to transmit screen captures over a local network using TCP Sockets. (Bu proje, ağ programlamanın temellerini anlamak için tasarlanmış eğitici bir prototiptir. TCP Soketlerini kullanarak yerel bir ağ üzerinden ekran görüntülerini iletmek için İstemci-Sunucu bağlantısı kurar.)

Technical Highlights (Teknik Özellikler):

Language: C# (.NET Framework/Core)

Networking: Uses System.Net.Sockets for TCP communication.

Mechanism: Captures desktop screenshots and serializes them for network transmission.

Architecture: Simple Client (Viewer) and Server (Host) logic.
